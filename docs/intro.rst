Introduction
============

This is the documentation for codingame, a library for Python to aid
in creating applications that utilise the undocumented CodinGame API.

Prerequisites
-------------

codingame works with Python 3.5 or higher.


.. _installing:

Installing
-----------

You can get the library directly from PyPI: ::

    python3 -m pip install -U codingame

If you are using Windows, then the following should be used instead: ::

    py -3 -m pip install -U codingame
