Pythonic wrapper for the undocumented CodinGame API.

Installing
----------

**Python 3.5.3 or higher is required**

Run the following command:

.. code:: sh

    # Linux/macOS
    python3 -m pip install -U codingame

    # Windows
    py -3 -m pip install -U codingame
